-----------------------
Super Cart Instructions
-----------------------

1. Download the project folder.

2. Install the Chrome extension:
			a. Open Chrome Extensions
			b. Turn On Developer mode
			c. Select 'Load unpacked'
			d. Select 'Chrome_ext' folder

	
* It is highly recommended to pin the Super Cart extension to the toolbar.

			
<---- Please download the extension before opening the E-commerce websites ---->


3. Open the demo E-commerce websites: "https://super-cart-shopping-site-1.netlify.app/", "https://super-cart-shopping-site-2.netlify.app/".

4. Open the Super Cart website by clicking the extension logo.

5. Add products to the different carts.

6. Enjoy viewing your products on the Super Cart! Feel free to sort/filer them as you like.
